🚀 Bootstrap Accordion

O Accordion é um componente que abre e fecha conteúdo (tipo sanfona).

Ele serve principalmente para organizar informações sem poluir a tela.
Exemplos reais:
❓ FAQ (Perguntas frequentes)
📋 Detalhes de um serviço
📚 Conteúdos longos (explicações, termos, instruções)
📱 Interfaces mobile (economizar espaço)

1. 👀 Organização visual
👉 O usuário vê só o que quer.

2. ⚡ Melhora a experiência do usuário (UX)
Fica mais fácil de navegar e entender o site.br
👉 Menos confusão = mais profissionalismo

3. 📱 Ideal para celular
Em telas pequenas, espaço é tudo.
👉 Accordion ajuda a “compactar” o conteúdo

4. 💼 Padrão de mercado
👉 Muito usado em:
👉 Sistemas web
👉 Sites institucionais
👉 Landing pages
👉 Plataformas de ensino
